服务器问题可在群内反馈 比如有人烧家、服务器崩了什么的

荒野求生多层档养老，玩家存档永不重置

多层世界设定为玩家天数小于100天只能在萌新天堂（地面）、萌新地洞、冒险世界、及角斗场之间跳跃，超过100天后可在所有世界之间跳跃

推荐我们服务器的直连模组，创意工坊搜索：荒野求生直连

服务器模组说明

服务器做了如下修改，请仔细阅读，具体如下：

  • 禁止投票回档和重置世界的功能

  • 只有萌新天堂才能换人

  • 只有迷你家园和角斗场不能读鸟书或者鱼书，其他世界只能在凌晨01:00至07:59之间才能无限读书，其他世界读书会有20s的冷却时间

  • 独一无二的物品不能放入背包

  • 小偷包锁

    • 小偷包背在身上时，私聊输入 +lock 可以锁定小偷包，其他玩家无法背起上锁的小偷包，私聊输入 +unlock 可以解锁小偷包
    • 小偷包背在身上时，私聊输入 +share KU_XXXXXXXX （需要将 KU_XXXXXXXX 替换成你要分享的玩家的克雷ID，且该玩家要和你在同一个世界），可以分享小偷包，你分享的玩家可以背起该小偷包，私聊输入 +unshare KU_XXXXXXXX 可取消分享
    • 小偷包背在身上时，私聊输入 +showshares 可以查看分享给了哪些玩家
    • 空的小偷包即使上锁别人也能背起来
    • Alt + 鼠标左键 点击地上的小偷包可以查看其主人

  • 物品破坏都会有宣告和后台记录，均可查

  • 每个容器（箱子、龙鳞箱子、冰箱、烟盒、蘑菇灯等）内物品变化后会记录，详见 箱子历史记录

  • 服务器自动清理：角斗场外都有清理功能

  • 萌新天堂

    • 死亡复活无生命值上限下降的惩罚，
    • 狗最多为每人3只

  • 灯泡和启迪碎片在第一次被捡起后会上锁，以后只有第一次捡起的玩家才能捡起

  • 在地下如果远古的钥匙因为bug消失了，可以私聊输入 +key 获取

  • 收船功能

    • 在船上放船舵后右击即可将船放入背包
    • 将船放到物品栏的第一个然后私聊输入 +rename 船名即可重命名你的船

  • 双击牛两下可将牛回收为牛角，吹牛角后再次召唤牛

  • 眼球塔之间不会互相攻击

  • 初始物品：以免冬夏进来的小伙伴惨死

  • 出了冒险世界和角斗场，其他世界亮茄会直接死

  • 迷你家园

    • 每人有一个免费的迷你家园
    • 进入迷你家园世界后，通过影灯可以打开或关闭你的个人家园
    • 每人的家园最多只能有1000个实体(实体为直接在地面上可以互动的物体，容器里的物品不计入实体)，超过1000个实体会随机删除至1000个
    • 为了防止卡顿，迷你家园会自动删除猪人、兔人、鱼人、蜜蜂、杀人蜂、蜘蛛
    • 具体见下面介绍视频

  • 角斗场 有挑战赛（可以获得金币奖励，每天五次）及PVP模式

    • 开始角斗场需要入场券，入场券获得方式为在线时间：每在线1分钟可以获得0.016667张入场券，在挂机天堂不会获得入场券
    • 入场券的数量通过游戏公告（右下角的按钮）查看
    • 龙鳞箱子可上锁：站在箱子旁在聊天窗口输入 +lockc 可锁定箱子（只有你能打开，有效期 120min，每次输入有效期自动延迟到 120min），输入 +unlockc 可解锁箱子，按 Alt + 鼠标左键点击箱子 可以显示箱子主人

  • 除了挂机天堂和角斗场外，其他世界人数超过8人（冒险世界和暗黑历险是4人）时，会触发传送机制，480s后传送天数最高的人到角斗场，防止人数超标

  • 玩家交易系统，详情请看游戏内说明（游戏商店面板的提示按钮）

  • 玩家私人仓库，详见仓库说明

  • 怪物击杀公告、猎犬来袭提示和蚁狮发怒提示功能

  • 无草蜥

  • 地洞入口无蝙蝠

  • 芦苇不可燃

  • 包裹显示：按 Alt + 鼠标左键 可显示包裹内容

  • 自动堆叠

  • 玩家砸建筑和烧东西提示（防熊）

  • 禁用投票重置世界

  • 管理者：某些常驻玩家拥有回档、封禁权限，帮忙管理服务器

  • 杀普通的无眼鹿有50%的概率掉落鹿角

  • 扫把可以换花及盆栽的样式

  • 木牌用锤子砸了之后可以变成打人的木牌

  • 眼球塔之间不会再互相攻击

  • 驯牛系统，详见视频

  • 骑牛下线：如果玩家下线时骑着牛，并且牛的驯服度>30%，牛会自动变成牛角，放到玩家的物品栏中

  • 回收船，详见视频

    • 玩家身上只能最多带两个船
    • 船不能放入切斯特、哈奇和沃比中

角斗场挑战赛说明

分为单人模式和双人模式，单人模式怪物血量是正常的一半，伤害是正常值的1.5倍，双人模式怪物的血量是单人的两倍，怪物的攻击是单人的1.5倍

每过一关就会获得对应的金币奖励，一共15关，奖励的金币分别为 2 2 2 2 3 2 2 2 2 3 2 2 2 2 20

双人模式当有一个玩家死亡，下一关会自动回落到单人模式，并且死亡的玩家不会获得奖励

角斗场PVP说明

开启PVP的方式

在游戏内聊天窗口输入：

/pvp on：开启 PVP 模式
/pvp off：关闭 PVP 模式

按组合键：

左Ctrl+Home：把一个地皮范围内的所有玩家随机分为两组
左Ctrl+Insert：把一个地皮范围内的所有玩家都加为好，机按 Ctrl + F 将不会再攻击这些玩家
左Ctrl+Delete：清空所有好友

注意：好友列表不会保存，每次重进游戏或者跳世界会自动清空列表

PVP机制

  • 在远古灯柱围成的正方形区域内才能攻击其他玩家
  • 右击影灯开始每局决斗，决斗开始时，会在四个角落生成远古宝箱，箱子每隔 30s 会随机刷新物资，刷新物资的数量根据场地内的玩家数
  • 再次右击影灯会提前解锁比赛并清空场地内物品
  • 决斗开始3分钟后会在周围生成一圈极光（无法作祟），5 分钟往内再生成一圈（类似吃鸡游戏的缩圈机制），7 分钟时远古宝箱会消失。另外比赛过程中场地内会随机生成多汁浆果丛
  • 进入场地的方法是通过虫洞，玩家进入虫洞前会自动掉落身上所有装备，请提前放到外围箱子里
  • 玩家进入场地后会获得一个草甲和长矛，玩家不管什么角色，其三维会被设定到 200/200/200，并加满三维
  • 进入虫洞后所有食物附加buff，如羊角冻和糖豆的 buff，都会取消
  • 进入虫洞后获得 30s 的无敌时间
  • 场地内地上的物品 30s 会自动消失
  • 该世界内懒人法杖，紫法杖，懒人护符会失效，恶魔的灵魂会自动消失
  • 作祟木牌可以直接复活
  • 该 PVP 世界内没有影怪
  • 角色平衡：
    1. wx78无加速效果，死亡后升级及过载时间均保留
    2. 女武神攻击变为1倍，无减伤，攻击无加血
    3. 鱼妹可以吃肉，女武神可以吃素，恶魔人吃东西效果不减半
    4. 虫木和火女的火焰伤害和其他角色一样，火女的冰冻伤害和其他角色一样
    5. 维斯和温蒂的攻击变为1倍
    6. 全角色都可读书
  • 物品平衡
    1. 排箫使用数次为2次，睡眠时间3s
    2. 睡书使用数次为1次，睡眠时间3s
    3. 旋风的耐久为4次
    4. 触手30s会自动清除

仓库说明

私人仓库：

开启方法：游戏内右小角点击仓库按钮，在弹出的面板中点击充值按钮即可消耗300游戏内的金币购买30天(现实)的仓库使用时间，如果金币不够，可以联系服主用现金购买，现金购买的价格为30元/半年，或者50元/年

仓库到期后仓库内的物品不会删除，只是冻结，无法取出及放入，续期后可继续使用

只有靠近地洞才能打开仓库面板，要将东西放入仓库，请使用魔法一本

服务器清理

清理机制

每10天检查一次地面的物品。地面的物品第一次被检查，会添加标记；被添加标记的物品在第二次检查时会被清理。这意味着地面物品至少10天以上才会被清理。

只会清理地面且不包含在白名单的物品，另外散落在海上的物品都会被清理

玩家身上及箱子里的，白名单里的，或者茶几附近（要很靠近）的物品不会被清理

  • 小动物：兔子、鼹鼠、萤火虫之类

船清理

船如果连续540天（游戏天数，即现实3天）无人踩上去过，就会被清理

回档、踢人及封禁权限

使用方法

聊天窗口输入如下指令 (不是控制台)

📌 保存并重启 +reset

用于模组修改后软重启

📌 回档 +rollback数字

举个例子，比如往前回2天就用+rollback2

+rollback回到当天开始时

+rollback1回到昨天开始时

数字最大为5

📌 杀人 +kill名字

把名字替换为要杀的玩家的名字的一部分，不用全名，只需名字的关键字，比如要封禁辣椒小皇纸，输入+kban辣椒或者+kban小皇纸或者+kban皇都可以，但是可能封掉名字也包含关键字的人

📌 踢人 +kick名字

方法同上

📌 封禁 +ban名字

方法同上

📌 先掉落所有物品再封禁 +kban名字

🚫 注意

命令不要加额外空格或者其他字符，否则无法识别

🚫 特别强调

如果发现因发生口角或其他私人矛盾而踢人封禁别人的，一律收回权限，请勿滥用权限
